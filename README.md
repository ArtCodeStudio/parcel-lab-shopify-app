# parcel-lab-shopify-app

## Description

Sync your Shopify orders to [parcelLab](https://parcellab.com).

## Installation

```bash
npm install
cp .env.example .env
# Now open and edit .env
```

## Running the app

```bash
# development
npm run start

# watch mode
npm run start:dev

# production mode
npm run start:prod
```

## Test

```bash
# unit tests
npm run test

# e2e tests
npm run test:e2e

# test coverage
npm run test:cov
```
